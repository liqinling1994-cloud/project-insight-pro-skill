# 03-证据与推广红线规则

## 证据等级

### 高可信

- 官方招生简章、合同、白皮书、项目手册
- 主管部门或行业监管平台
- 学校、机构、品牌官网
- 招生办公室或项目办公室公开文件
- 权威数据库、排名官网、备案系统
- 用户提供的一手内部资料

默认置信度：A。资料过期或字段含糊时降级。

### 中可信

- 主流媒体报道
- 行业报告
- 公开讲座资料
- 有清晰身份的从业者分析
- 多个平台互相印证的公开信息

默认置信度：B。缺时间或来源单一时降级。

### 低可信

- 匿名评论
- 无来源搬运
- 营销软文
- 内容农场文章
- 发布时间不明的信息
- 没有上下文的截图

默认置信度：C 或 D。低可信来源不能支撑强卖点。

## 证据账本字段

| 字段 | 说明 |
|------|------|
| 信息编号 | 对应 `BAS-F`、`ESS-A`、`POS-R` 等编号 |
| 信息点 | 被使用的事实或判断 |
| 来源 | 链接、文件名、页码、截图编号或用户输入位置 |
| 来源类型 | 官方、媒体、用户、一手口碑、二手观点、推导 |
| 时间 | 发布、获取或适用时间 |
| 来源可信度 | 高、中、低、待核实 |
| 置信度 | A/B/C/D |
| 使用位置 | 输入包、话术、后续写作 |
| 备注 | 风险、限制、复核建议 |

## 证据使用规则

- 无来源的信息必须标注“待核实”，置信度不得高于 C。
- 搜索不到信息时写“未找到可复核来源”。
- 排名、分数线、费用、政策必须带年份或适用周期。
- 过期信息只能用于背景，不可用于强结论。
- 口碑类内容不得冒充官方事实。
- C 级信息需要人工确认后才能用于宣传。
- D 级信息只能用于内部判断或补证清单。
- 强卖点必须有 A/B 级证据支撑。

## 推广红线

### 绝对禁止

- 保录取、包录取、保毕业、保拿证
- 保就业、保证高薪、保证进名企
- 100% 通过、必上岸、稳赚
- 贬低其他学校、项目、机构或国家
- 将未经核实的口碑写成事实
- 将官方宣传包装成学生真实反馈
- 使用过期排名制造当前优势
- 暗示主管部门认可但无可复核证据
- 把 C/D 级信息写成确定事实

### 高风险表达处理

| 高风险表达 | 替代表述 |
|------------|----------|
| 保证录取 | 根据公开要求，符合条件的学生可重点关注 |
| 保证拿证 | 学生需完成课程和考核要求后获得相应证书 |
| 必进名校 | 有机会申请更高层次院校，结果取决于成绩、材料和当年政策 |
| 比普通本科强太多 | 适合看重国际化路径的家庭，选择前需比较费用、目标和风险 |
| 国家认可 | 需以主管部门公开平台或官方文件为准 |
| 低分逆袭名校 | 适合分数和目标不完全匹配、希望增加路径选择的家庭 |
| 稳赚不亏 | 费用、时间和结果均存在不确定性，需结合家庭条件判断 |

## 输出前检查

- 是否有绝对化承诺。
- 是否有缺来源的强结论。
- 是否有收益、录取、就业、证书结果保证。
- 是否有未经授权的具体案例。
- 是否有误导性模糊表达。
- 是否把 C/D 级信息用于外部宣传。
- 是否绕过了 `POS-W` 风险编号。

发现风险时，必须标注风险编号并给出替代表述。
