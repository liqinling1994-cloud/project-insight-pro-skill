# 项目洞察助手PRO 中文介绍

## 一句话说明

项目洞察助手PRO 用来把原始项目资料拆解成可读、可判断、可交接的项目宣传写作输入包。它服务项目宣传前的第一阶段，让内容、销售、咨询和后续写作流程先拿到清楚的业务判断。

## 适合谁使用

- 教育咨询团队。
- 课程、训练营、服务产品团队。
- AI 工具、内容产品和复杂产品推广团队。
- 负责项目宣传、内容选题、销售转化的运营人员。
- 需要把复杂资料拆成推广输入包的顾问或负责人。

## 输入什么

可以输入：

- 招生简章、项目手册、课程页、官网介绍。
- 用户口述、顾问笔记、内部资料。
- 竞品资料、替代方案资料。
- 截图文字、文章摘录、政策文件。
- 销售话术、客户问答、公开案例和历史宣传稿。

资料不完整时，skill 会继续做有限分析，并列出最小补证清单。

## 输出什么

默认输出“干净阅读版”项目洞察报告，包含：

- 产品一句话定义。
- 用户实际购买的东西。
- 交易与交付结构。
- 核心价值与结果边界。
- 适合人群和不适合人群。
- 用户决策链和成交阻力。
- 日常推广人群、内容资产和转化入口。
- 下游写作指令。
- 需要补证的材料。

默认不展示编号、置信度和证据账本。只有明确要求审计版时，才追加编号、置信度、来源链和证据账本。

## 运行逻辑

项目洞察助手PRO按五步运行：

1. 识别项目：确认主体、交付物、价格、目标结果和资料缺口。
2. 建立底稿：记录事实、来源、时效、风险和待核验点。
3. 拆交易交付：说明用户买到什么、怎么交付、谁负责、边界在哪里。
4. 判断人群：识别适合对象、付款者、使用者、影响者和反对者。
5. 输出交接：给出推广主线、内容资产、转化入口和下游写作指令。

## 推广红线

skill 内置证据与推广红线规则，会主动识别高风险表达，例如：

- 保录取、包录取、保毕业、保拿证。
- 保就业、保证高薪、保证进名企。
- 100% 通过、必上岸、稳赚。
- 将未经核实的口碑写成事实。
- 使用过期排名制造当前优势。
- 保证爆款、成交、增长。

发现风险时，报告会给出更稳妥的替代表述。
