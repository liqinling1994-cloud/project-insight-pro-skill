项目洞察助手PRO

把原始项目资料拆解成干净、可读、可判断、可复用的项目洞察报告，帮助团队先看清产品、交易、交付、人群、风险和推广打法。

当前版本：1.4.0

默认输出干净阅读版，不展示编号、置信度和证据账本。审计版需要明确提出。

它解决什么

看清项目

拆清产品定义、交付物、交易结构、费用边界和结果路径。

判断人群

识别适合人群、不适合人群、购买者、使用者、影响者和反对者。

交接推广

输出日常推广人群、内容资产、销售转化入口和下游写作指令。

默认输出

  • 一页结论。
  • 产品拆解。
  • 交易与交付结构。
  • 价值判断与结果边界。
  • 目标人群与成交阻力。
  • 日常推广打法和内容资产。
  • 下游写作指令。

推广红线

  • 不写保录取、包录取、保毕业、保拿证。
  • 不写保就业、保证高薪、保证进名企。
  • 不把未经核实的口碑写成事实。
  • 不使用过期排名制造当前优势。

快速安装

把 skill/项目洞察助手PRO/ 整个目录复制到本地 skills 目录。

~/.codex/skills/项目洞察助手PRO/

推荐调用

启动项目洞察PRO，基于下面的项目资料，输出一份项目洞察报告。
启动项目洞察PRO，输出审计版，包含编号、置信度、来源链和证据账本。

仓库入口

运行文件位于 skill/项目洞察助手PRO/。安装、说明、示例和发布校验文件放在仓库根目录、docs/、examples/ 和 scripts/。
